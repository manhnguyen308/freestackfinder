# Factual Freshness Checks

Use this checklist when refreshing existing articles. It catches facts that can become stale even when `lastmod` is recent.

## Content That Ages Fastest

- AI tools: model names, free message limits, file upload support, web access, image credits, coding features, privacy controls, and plan names.
- SaaS free plans: user seats, project counts, automation runs, storage, exports, watermark behavior, file size limits, and support access.
- Creative tools: asset libraries, commercial-use rules, background removal, brand kits, AI credits, export formats, and team collaboration limits.
- Cloud and storage: free GB amounts, shared-storage rules, transfer quotas, version history, device sync, file upload caps, and encrypted-storage claims.
- Security tools: device limits, audit status, VPN data caps, password vault sharing, 2FA sync behavior, and antivirus lab-test claims.

## Monthly Checks

- Scan published content for model names, exact prices, exact plan limits, "currently", "latest", "new", "unlimited", and "as of" claims.
- Prioritize high-impression pages, AI/model pages, cloud storage pages, business SaaS pages, security pages, and recently refreshed pages with precise claims.
- Verify volatile facts against official pricing pages, official help docs, product release notes, or current product documentation.
- If official sources are unclear or inaccessible, remove the exact number and use reader-safe wording that tells users what to verify.
- Update `lastmod` to the current local date only for articles changed.

## Claim Checklist

Before keeping a precise claim, confirm:

- Product name and plan name still exist.
- Free plan still has the stated feature.
- Usage, storage, export, upload, transfer, message, or seat limit is still current.
- Paid-plan comparison is still accurate.
- "Unlimited" is qualified by fair-use, platform, region, or plan caveats when relevant.
- Model names are still available in the product surface being discussed.
- Pricing is region-aware or clearly framed as a current US list price.
- Commercial-use, licensing, privacy, and audit statements are backed by current official docs.

## Evergreen Wording

Prefer wording that survives plan changes:

- "check the current free-plan limit before switching"
- "free limits change often"
- "paid plans raise limits and add workflow features"
- "verify this feature is still included before committing"
- "the main decision point is..."

Avoid anchoring a recommendation to one exact model name, storage number, monthly credit count, or price unless that detail is both verified and essential to the article.
